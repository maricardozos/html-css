Como a internet funciona?

O que é bit e byte?

O computador é capaz de compreender duas coisas: 0 e 1.

  • Um bit é a menor unidade de informação em um computador, representando um valor binário de 0 ou 1.
  • Um byte é um grupo de 8 bits, que juntos podem representar 256 valores diferentes (de 0 a 255). Bytes são usados para armazenar dados com caracteres, números e outros tipos de informação em um computador.

O que é MB e Mb?

1 byte = 8 bits, então 1 MB ≠ 1 Mb.

  • MB (megabyte) é uma unidade de medida de armazenamento de dados, equivalente a 1.024 kilobytes (KB) ou aproximadamente 1 milhão de bytes.
  • Mb (Megabit) é uma unidade de medida de transferência de dados, equivalente a 1.024 kilobits (Kb) ou aproximadamente 1 milhão de bits.

Roteadores

Os pacotes de dados podem chegar em seu PC por diversas rotas diferentes, tudo vai depender do tráfego no momento da transmissão.

Identificando os nós

Uma das funções do TCP/IP é identificaros nós. Um nó é cada ponto que está conectado à rede. Quando voc~e "se conecta" à internet, recebe uma identificação única. Essa identificação é um endereço IP.Alías, os números IP mudam constantemente.

DNS (Domain Name System)

Traduz nomes de domínios (como www.exemplo.com) em endereços IP, permitindo que os navedores encontrem e acessem sites na internet.

Domínio e hospedagem

Quando você acessa um site, precisa da URL. A parte principal dessa URL é o domínio.

  • GTLD: TLDS genéricos (.com, .net, .gov ...)
  • ccTLD: TLDS com designação do país (country code). Exemplo: .com.br, .edu.us, .jp.

Obs: uma URL pode ter http:// ou https://

  • HTTP é um protocolo de transferÇencia de dados sem segurança.
  • HTTPS é a versão segura com criptografia, protegendo as informações transmitidas entre o navegador e o servidor.

Para manter seus arquivos acessíveis (quando você desenvolver seu site), você vai precisar contratar um plano de hospedagem que normalmente oferece:

  • Espaço em disco
  • Memória
  • Banco de dados
  • Contas de e-mail
  • Sub-domínios
  • Certificados de seg.
