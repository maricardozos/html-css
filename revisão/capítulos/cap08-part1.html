Exercício 008

Principais formatações

Negrito / Destaque

Nessa frase, tenho um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo usando as tags <big> e <small>. Obs: o big já está obsoleto, então devemos evitar ele.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Para isso utilizamos a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. Para isso usamos a tag <ins>

Existe também o sublinhado com a tag u (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3 utilizamos a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O utilizamos a tag <sub>
